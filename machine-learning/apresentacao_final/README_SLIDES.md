# Slides sobre Transformers

Este diretório contém apresentações sobre Transformers em diferentes formatos.

## Arquivos Disponíveis

### 1. `slides_transformers.md`
- Formato Markdown compatível com **Marp**, **Reveal.js**, e outras ferramentas
- Pode ser convertido para PDF, PowerPoint, ou HTML
- 19 slides cobrindo todos os aspectos dos Transformers

### 2. `slides_transformers.html`
- Apresentação HTML interativa
- Pode ser aberta diretamente no navegador
- Navegação com botões ou teclado (setas ou espaço)
- Design moderno e responsivo

## Como Usar

### Opção 1: HTML (Mais Simples)
1. Abra `slides_transformers.html` no seu navegador
2. Use as setas do teclado ou os botões para navegar
3. Pronto para apresentar!

### Opção 2: Marp (Para PDF/PPT)
1. Instale o Marp: https://marp.app/
2. Abra `slides_transformers.md` no Marp
3. Exporte para PDF ou PowerPoint

### Opção 3: Reveal.js
1. Instale Reveal.js: `npm install -g reveal-md`
2. Execute: `reveal-md slides_transformers.md`
3. Apresente no navegador
